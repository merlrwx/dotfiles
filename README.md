# Terminal Dotfiles

Essential terminal configuration for bash, vim, tmux, and alacritty.

## Setup Instructions
```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply git@github.com:merlrwx/dotfiles.git`
```
